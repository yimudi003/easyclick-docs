## 工程结构说明
- build: 编译后的目录，编译的apk和iec会存储此处
- src: 源码文件夹
    - js: 存放js源码的地方，系统会自动加载
    - layout: UI文件存放的文件夹
    - plugin: 插件文件夹
    - res: 图片和部分资源存储文件夹
    - update.json: 热更新配置文件夹
