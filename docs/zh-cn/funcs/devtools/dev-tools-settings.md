## 基础设置
    - <img src='/androidimg/setting/basic_set.png' alt="basic_set.png" style={{zoom:'30%'}} />
    - `调试版本包名`，配合打包调试版本apk使用，这里写打包调试版本apk，用来识别调试app的

    - `激活器密码和激活器端口`，需要配合激活器使用，一般不要修改
