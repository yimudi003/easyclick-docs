```text
# 4.1.0

```
:::tip
- 无限制访问的token，用于全站最高的api权限
- 默认是空代表无效，如果要设置 请设置一个大于16位的复杂字母+数字组合
- 在请求的时候加上http头，例如 header["no_limit_token"] = "xxxxx" 即可
:::
